## 一、RackNerd 是什么来头？

简单说，RackNerd（简称 RN）是一家美国 IaaS 服务商，主要卖 KVM VPS、独立服务器、虚拟主机和分销主机。创始人 Dustin B. Cisneros 是行业老人，2019 年创立公司，靠着价格低、配置高、服务稳这三板斧，快速积攒了大批忠实用户。

它家的 VPS 有几个标配特点：KVM 虚拟化、SSD RAID-10 阵列、1Gbps 带宽、1 个独立 IPv4、SolusVM 控制面板，下单即开通。

数据中心横跨美国东西海岸、加拿大、欧洲，目前一共 21 个机房，包括：洛杉矶 DC-02/DC-03、圣何塞、西雅图、达拉斯、芝加哥、纽约、阿什本、亚特兰大、多伦多、阿姆斯特丹、法国斯特拉斯堡等。

说到国内用户，RN 提供的是标准国际线路，并非 CN2 GIA 专线，延迟大约在 150–220ms 之间，西海岸机房（洛杉矶/圣何塞/西雅图）对中国大陆的访问速度相对最优。
